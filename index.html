Exploitable Linux Configurations

May 2019 | Group Project

I worked in a team to create an OSSEC-like intrusion detection system to detect unauthorized file system changes and generate a log of malicious activity on Linux systems. Using a modular design, we also implemented a change prevention and rollback mechanism, rootkit detection strategies, and a customizable activity timeline.

We presented on our utility during the RIT Computer System Forensics class' binary analysis week. Read the report, check out the tool on GitHub, or view our presentation below.

Rogue Network Devices

December 2018 | Honors Project

To expand on the topics covered in my Network Services class for the RIT Honors program, I identified common security vulnerabilities and oversights that would allow an attacker to take control of a network using common services such as DHCP, DNS, and wireless access points.

Then, I simulated rogue devices on a network, determined how to effectively detect and isolate them, and learned about technologies to increase network security.
